Pierre-Feuille-Ciseaux

Mon premier jeux-vidéo !

Les règles du pierre-feuille-ciseaux sont très simple. Le joueur choisi entre :

  • La pierre : qui écrase les ciseaux
  • La feuille : qui emballe la pierre
  • Les ciseaux : qui découpe la feuille
Et l'ordinateur choisi aussi entre ces trois objets. On vous propose d'afficher le résultat une fois les choix effectués. Et éventuellement de garder les scores en mémoire.

Votre programme peut être en console (terminal) ou plus visuel !

Pour les systèmes un peu exotique, n'hésitez pas à rédiger un README pour expliquer comment "builder" et "lancer" votre petite application !
