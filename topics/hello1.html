Hello World

Mettre en place sa première page web

Découvrir un outil (langage ou framework) pour faire une application web implique souvent de devoir apprendre à réaliser une première page. Pour cet atelier, on vous propose de réaliser un projet qui se contente d'afficher "Hello World" à l'écran. Cet exercice servira de base à des exercices suivants.

Pour les systèmes un peu exotique, n'hésitez pas à rédiger un README pour expliquer comment "builder" et "lancer" votre petite application !
