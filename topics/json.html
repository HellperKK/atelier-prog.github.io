Mettre en forme un fichier JSON externe

Initiation aux webservices

De plus en plus d'applications web exposent des web-services, permettant à d'autres applications (ou logiciels) d'utiliser leurs contenus (ou d'exécuter des actions), dans cet exercice, on vous propose de créer une page web qui met en forme les données de ce fichier : products.json. Donc de charger le contenu du fichier et de l'afficher dans une autre application.

Pour les systèmes un peu exotique, n'hésitez pas à rédiger un README pour expliquer comment "builder" et "lancer" votre petite application !
